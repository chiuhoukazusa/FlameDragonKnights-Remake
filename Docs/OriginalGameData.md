## 🎮 核心玩法机制

### 地图系统
- **格子类型**: 六边形网格
- **地形种类**:
  - 草地：移动消耗 1，无防御加成
  - 森林：移动消耗 2，防御 +10%
  - 山地：移动消耗 3，防御 +20%
  - 城堡：移动消耗 1，防御 +30%
  - 水域：不可通行

### 战斗系统
- **回合制**: 玩家回合 → 敌人回合
- **行动顺序**: 由速度值决定（同一回合内）
- **攻击类型**:
  - 物理攻击：近战武器
  - 远程攻击：弓箭、投掷
  - 魔法攻击：法术

## 💰 经验值与等级系统

### 经验值获取
- 击败敌人: 敌人等级 * 10 + 基础经验
- 治疗队友: 治疗量 / 10
- 使用魔法: 5 经验

### 升级所需经验
```
Lv 1 → 2: 100 EXP
Lv 2 → 3: 120 EXP
Lv 3 → 4: 140 EXP
公式: 100 + (等级 - 1) * 20
```

## 🎯 战斗公式（推测）

### 物理伤害
```
基础伤害 = 攻击者攻击 - 防御者防御 / 2
地形修正 = 基础伤害 * (1 - 地形防御加成)
随机波动 = 0.9 ~ 1.1
最终伤害 = 地形修正 * 随机波动
```

### 命中率
```
基础命中 = 90%
命中率 = 基础命中 + (攻击者速度 - 防御者速度) * 2%
最终命中率 = max(30%, min(95%, 命中率))
```

### 暴击率
```
暴击率 = 攻击者幸运 / 2 - 防御者幸运 / 4
暴击伤害 = 伤害 * 2
```
